## My keyboard layout
US / RU

> [!WARNING]
> It doesn't work correctly in Niri because it's compiled with a version of libxkbcommon that doesn't support format v2.

Caps + LShift   => ISO_Next_Group
LShift + Caps   => ISO_Next_Group
LShift + RShift => Caps Lock
RShift + LShift => Caps Lock
LCtrl           => Hyper_L
Caps            => LCtrl
Menu            => Compose
Shift + Menu    => Menu
